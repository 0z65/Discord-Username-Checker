# Discord Username Checker Tool 

An advanced and high-speed tool for checking Discord usernames, featuring proxy support and multi-threading for maximum performance.

## ‚ú® Features
1. **Blazing Fast**: Supports multi-threading to perform thousands of checks in a short time (up to 2000 threads).
2. **Proxy Support**: Smart system for managing and rotating proxies to avoid Rate Limits.
3. **Multiple Generation Patterns**: Includes 13 different username generation patterns (3-char, 4-char, semi-rare, numbers, and more).
4. **Webhook Alerts**: Instantly send available usernames to your Discord server via Webhook.
5. **Auto-Save**: Automatically saves available usernames to `good.txt`.
6. **Flexible Configuration**: Easy setup for Proxies and Webhooks using the `.env` file.
